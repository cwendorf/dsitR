# [`dsitR`](https://github.com/cwendorf/dsitR/)

## Color agents based on values

### Description

Maps numeric values to colors using a color palette.

### Usage

```r
color_agents(values, col_palette = NULL)
```

### Arguments

- **`values`**: Numeric vector of values to map to colors.
- **`col_palette`**: Optional character vector of colors. If NULL, a blue-white-red palette of 100 colors is used.

### Value

Character vector of colors corresponding to the input values.
